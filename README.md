# Projeto-de-an-lise-de-dados-com-chatbot
O projeto analisa um dataset grande, com mais de 200 mil linhas. Responsável por conter informações sobre casos reais de covid 19 entre 2020 e 2021, o dataset faz parte de um projeto de análise de dados, onde após um tratamento de dados nulos e/ou desnecessários,  foi realizada uma análise exploratório das informações , a fim de explicar ocorridos.
